# cs_gstack — CS2 Skins Analysis System

## О проекте
Персональная аналитическая система для анализа предметов и лотов CS2.
Цель: принятие решений о покупке (buy / no-buy / manual review) на основе
item identity, exact-параметров, рыночной цены, ликвидности, community signals и risk flags.

## gstack
Use /browse from gstack for all web browsing. Never use mcp__claude-in-chrome__* tools.
Available skills: /office-hours, /plan-ceo-review, /plan-eng-review, /plan-design-review,
/design-consultation, /review, /ship, /land-and-deploy, /canary, /benchmark, /browse,
/qa, /qa-only, /design-review, /setup-browser-cookies, /setup-deploy, /retro,
/investigate, /document-release, /codex, /cso, /autoplan, /careful, /freeze, /guard,
/unfreeze, /gstack-upgrade.

## Стек
- Python 3.12+
- httpx для HTTP-запросов
- SQLite для локального хранения
- Pydantic для моделей данных
- Rich для CLI-вывода
- pytest для тестов
